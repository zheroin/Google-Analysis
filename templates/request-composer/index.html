Overview

The purpose of the Request Composer is to illustrate a small subset of the {{site.tech.reporting_api_v4.name}} features.

  • Histogram buckets
  • Pivot requests
  • Cohort requests

To simplify the processing of the API responses, this tool applies some additional limitations on the requests that can be made. For full specifications of the API see the reference documentation. There are other features of the Analytics Reporting API V4, such as metric expression, multiple date ranges, and advanced filter configuration which are not demonstrated with the current version of the Request Composer, however we accept pull requests.
